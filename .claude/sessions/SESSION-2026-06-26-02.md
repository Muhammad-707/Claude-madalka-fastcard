## Решения
- **204 No Content для getProducts:** `responseHandler` в query config, возвращает пустой Paginated если status 204.
- **График без time-series API:** deterministic PATTERN array × API totals → 12 точек на 3 dataKey. dataKey меняется при клике на карточку.
- **Клиентская фильтрация:** role filter (Users) и stock filter (Products) применяются к текущей странице, т.к. API не поддерживает эти параметры.
- **COLOR_MAP:** строковый маппинг имён цветов → hex, substring matching через `getColorHex()`.
- **ProductEdit reuses Create:** `ColorSelector` и `ImageUploader` импортированы из `@/pages/product-create/components/`.
- **Brands inline edit:** вместо модалки — inline Input в строке списка, Enter/Escape поддержаны.
- **Banners без API:** полностью статичный design mock с drag&drop зоной, сохраняет файлы только в local state.

## Блокеры / вопросы
- Banners API отсутствует в Swagger — страница static, готова к подключению когда появится.
- Subcategories endpoint (`/Category/get-sub-categories`) требует `categoryId` — всё завязано на выбор категории в форме товара.
